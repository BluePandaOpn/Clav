# Documentacion del Proyecto

Esta carpeta contiene la documentacion principal para entender, mantener y auditar todo el codigo de este proyecto.

## Archivos de esta carpeta

- `docs/SISTEMA_COMPLETO.md`
  - Explica arquitectura, flujo funcional, modulos, datos, seguridad, desarrollo y operacion.
- `docs/AI_SECURITY_REVIEW.md`
  - Guia para que una IA o un auditor humano revise vulnerabilidades y calidad del codigo de extremo a extremo.

## Inicio rapido

1. Leer `docs/SISTEMA_COMPLETO.md` para entender como funciona todo.
2. Ejecutar la auditoria con `docs/AI_SECURITY_REVIEW.md`.
3. Registrar resultados y fixes en un PR con severidad (`Critical/High/Medium/Low`).
